# wp-plugin-blocks-starter

## usage

### get this repo

```bash
$ degit lotwt/wp-plugin-blocks-starter
```

### add a new block

```bash
$ cd src
$ pnpm dlx @wordpress/create-block --no-plugin
```

### generate a zip

```bash
$ pnpm plugin-zip
```
